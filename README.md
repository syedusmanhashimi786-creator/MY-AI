# StudyGen AI

**StudyGen AI** is an interactive AI chatbot designed to help students learn efficiently. Powered by OpenAI's GPT-3.5 API, it provides answers, explanations, and study guidance in a friendly and accessible way.

---

## 🎯 Features

- Modern chat interface with **chat bubbles**  
- Typing animation for realistic responses  
- Custom **brain + lightning logo**  
- Mobile-friendly and responsive design  
- Easy to customize AI name, logo, and theme  
- Built with **HTML, CSS, and JavaScript**  
- Connects directly to OpenAI API (or via backend for safety)

---

## ⚡ How to Run

1. Clone or download the project folder.  
2. Place your **API key** inside `index.html` or a secure backend.  
3. Make sure the logo file (`brain-lightning.png`) is in the same folder as `index.html`.  
4. Open `index.html` in a modern web browser.  
5. Type your question and press **Send** to interact with StudyGen AI.

---

## 🛡️ Safety Note

- **Do not expose your API key publicly** when hosting online.  
- For safe deployment, use a **backend server** to handle API requests.  

---

## 🛠️ Customization

- **Change AI Name:** Update `<h2 id="ai-name">StudyGen AI</h2>` and the system prompt in JavaScript.  
- **Change Logo:** Replace `brain-lightning.png` with your own logo.  
- **Theme Colors:** Adjust colors in the CSS section of `index.html`.
