# MY-AI
MY-AI is a smart AI assistant that helps users answer questions, learn new topics, and improve studying.

It can also generate creative ideas such as:
- Logos
- Slogans
- Website ideas
- Content

## Features
- AI question answering
- Study assistance
- Idea generation
- Simple and easy interface

## How to Use
1. Open the website
2. Type your question
3. The AI will generate an answer
